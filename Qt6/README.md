## 怎么编译本项目
### 依赖
- Qt6 [Qt官网](https://www.qt.io/)
- 开发工具 Qt Creator 或者其它
### 编译
- git clone 或 下载本项目源码
- Qt Creator打开 ETS2_KeyBinder.pro
- 执行qmake
- 编译运行
